# Ассистент-бот (Telegram)

Минимальный бот для двух функций: **расписание с напоминаниями** и **планирование поездок**.
Хранит данные локально в SQLite (`assistant.db`), создаётся автоматически при первом запуске.

## 1. Получить токен бота

1. В Telegram напиши **@BotFather**
2. Команда `/newbot`, следуй инструкциям (имя + username, должен заканчиваться на `bot`)
3. BotFather пришлёт токен вида `123456789:AAExxxxxxxxxxxxxxxxxxxxxxxxxxx` — сохрани его

## 2. Установка (локально или на сервере/VPS)

Нужен Python 3.10+.

```bash
# распакуй проект, зайди в папку
cd assistant_bot

# создай виртуальное окружение (рекомендуется)
python3 -m venv venv
source venv/bin/activate        # Windows: venv\Scripts\activate

# поставь зависимости
pip install -r requirements.txt
```

## 3. Токен

```bash
export BOT_TOKEN="твой_токен_от_BotFather"     # Windows (PowerShell): $env:BOT_TOKEN="..."
```

## 4. Запуск

```bash
python bot.py
```

В консоли появится `Бот запущен.` — теперь пиши боту в Telegram `/start`.

Чтобы бот работал постоянно (а не только пока открыт терминал), на VPS удобно
запускать его через `systemd`, `screen`/`tmux`, или `pm2`/`supervisor`. Простой вариант:

```bash
nohup python bot.py > bot.log 2>&1 &
```

## Команды бота

**Расписание**
- `/add_event 25.07.2026 14:30 Съёмка в студии` — добавить событие
- `/schedule` — список ближайших событий
- `/del_event 3` — удалить событие по id

Бот сам присылает напоминание, если до события остался час (проверка каждые 5 минут).

**Поездки**
- `/add_trip Дубай 25.07.2026 30.07.2026 Основная съёмка` — создать поездку (город, начало, конец, заметка — необязательна)
- `/trips` — список поездок
- `/trip 1` — подробности поездки и её маршрут
- `/add_stop 1 26.07.2026 09:00 Трансфер в отель` — добавить пункт в маршрут
- `/del_trip 1` — удалить поездку

## Важно про даты

Формат везде: `ДД.ММ.ГГГГ` для дат и `ЧЧ:ММ` для времени (24-часовой формат).

## Что можно добавить дальше

Это минимальный каркас под две функции, которые вы выбрали в первую очередь.
Дальше можно нарастить:
- Шаблоны сообщений брендам (`/template ...`)
- Контент-план и брифы
- Несколько чатов/пользователей с разграничением доступа
- Инлайн-кнопки вместо команд текстом
- Часовые пояса, если поездки в разные страны
